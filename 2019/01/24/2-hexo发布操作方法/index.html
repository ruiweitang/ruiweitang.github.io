hexo常见命令

  • hexo n '博客标题'创建一篇新的博客
  • hexo g 生成博客
  • hexo d 发布博客

博客内容格式设置（在md文件的最上方设置即可）

  • 上传到git的文件是public文件夹的内容（不需要管）
  • 自己的的md文件放到source/_posts文件夹中，注意：如果需要配置域名操作，需要在source文件夹中放CNAME文件（这个文件没有后缀名），内容写上自己的域名即可（ruiweitang.com）

上传到git的发布配置

安装hexo的部署工具npm install hexo-deployer-git --save

修改_config.yml文件

之后的操作非常简单了（ 每次需要操作的以下几步）

  1. 把写好的md文件放到\source\_posts 文件夹中，md文件中的图片路径记得需要改为从根目录开始\，把md文件中的图片放到images文件夹中
  2. 在总的根目录打开命令台，放好md文件之后，输入两个命令几个
    • hexo g：生成静态页面
    • hexo d：发布博客（把静态页面发送到git上）
